Acknowledgements

The Little Bird website is run using Django 1.9.
The sentiment analysis tools used come partially or fully from the Natural Language Toolkit .
The tweets are taken from Twitter using the Twitter API.
The word cloud was made using amueller's word cloud maker.
Rotten Tomatoes ratings were taken from OMDb (The Open Movie Database).
Goodreads ratings were taken from Goodreads API.
Pitchfork ratings were taken from Omar Delarosa's unofficial Pitchfork API.
